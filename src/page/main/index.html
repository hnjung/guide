이 가이드는 OOO 프로젝트에서 마크업 개발자가 알아야 할 모든 정보를 체계적으로 제공하는 문서입니다.

프로젝트 환경 설정, 해상도 기준, 폴더 구조 등 기본적인 작업 환경 설정부터 시작해, HTML, SCSS, JavaScript 코딩 규칙을 상세히 설명합니다. 또한 레이아웃 구성, 반응형 웹 디자인, 콘텐츠 요소(타이포그래피, 테이블, 카드 등)를 다루며, 성능 최적화를 위한 파일 압축 및 이미지 스프라이트 생성 방법도 포함되어 있습니다.

실제 개발에 유용한 다양한 컴포넌트(탭, 아코디언, 팝업 등)와 유틸리티(그리드, 폰트, 여백 처리 등)에 대한 가이드를 제공하며, 웹 접근성을 고려한 ARIA 속성, 리더기 지원 및 초점 관리 등을 포함해 접근성 향상에 기여합니다.

각 섹션은 프로젝트의 일관성과 효율성을 높이기 위한 실무적 예시와 설명을 통해 명확하게 구조화되어 있어, 개발자들이 쉽고 빠르게 참조할 수 있습니다.
