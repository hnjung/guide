폴더

  • /CSSPUB/view/ : 루트
  • /CSSPUB/view/css/ : 스타일시트
  • /CSSPUB/view/js/ : 스크립트
  • /CSSPUB/view/image/ : 이미지
  • /CSSPUB/view/html/ : HTML
  • /CSSPUB/view/xxx/com/ : 각 폴더의 공통 파일 (공통 스타일시트 예시 : /CSSPUB/view/css/com/)
  • /CSSPUB/view/xxx/xxx/ : 카테고리 별 파일 (상품 페이지 예시 : /CSSPUB/view/html/prd/)
  • /CSSPUB/view/doc/ : 산출물 목록 및 가이드(배포 대상 아님)

파일명

  • 스크립트 파일은 개발 파일과 구분을 위해 _pub를 파일명 맨 뒤에 추가 (예시 : common_pub.js)
