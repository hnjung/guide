가이드

변수사용

  • 디자인 가이드에 공통으로 잡혀있는 스타일은 가능한 변수로 선언하여 사용

이미지

64*64, -max

  • 아이콘 크기 : --icon-max : 6.4rem
  • 이미지 경로 : --bg-url-max : url('../../image/com/ico-common-max.svg');
  • 아이콘 사이즈 : --bg-size-max : 64.0rem auto;
  • 예시 : width: var(--icon-max); height: var(--icon-max); background: var(--bg-url-max) no-repeat 0 calc(var(--icon-max) * -1); background-size: var(--bg-size-max);

48*48, -xl

  • 아이콘 크기 : --icon-xl : 4.8rem
  • 이미지 경로 : --bg-url-xl : url('../../image/com/ico-common-xl.svg');
  • 아이콘 사이즈 : --bg-size-xl : 48.0rem auto;
  • 예시 : width: var(--icon-xl); height: var(--icon-xl); background: var(--bg-url-xl) no-repeat 0 calc(var(--icon-xl) * -1); background-size: var(--bg-size-xl);

웹접근성

리더기

  • 리더기에서 연결/해제 제어가 필요한 경우 해당 함수 사용 : commUiFnAriaHide();

초점

  • 초점 연결/해제 제어가 필요한 경우 해당 함수 사용 : commUiFnFocus();
  • 스크립트 공통 전역변수의 초점 요소 commUiVar.focusElements : 'a[href], area[href], input:not([disabled]), select:not([disabled]), textarea:not([disabled]), button:not([disabled]), [tabindex="0"], [data-focus-on="true"]'
  • 초점 요소가 아니고 숨기기/보이기 처리를 해야하는 항목의 경우 tabindex="0" data-focus-on="true" 코드 추가하여 제어 (예시 : 팝업 콘텐츠 DIV 요소 / iOS는 초점 하위의 첫번째 초점 요소에 초점이 잡힘)

숨기기/보이기 애니메이션

  • transition으로 처리 시 요소의 display:none;이 있는 경우 애니메이션 되지 않음
    display:none;을 사용하지 않고 height, overflow, visibility 등으로 처리할 경우 리더기와 초점 이동 관련 코드를 추가하여 액션 시 컨트롤 해야 하고, iOS의 경우 초점 대상이 아닌 요소의 제어가 어려움 (현재 가이드의 아코디언이 transition을 활용한 방법)
  • animation을 사용하면 display:none;이 있어도 동작 가능. 단, 숨기기 애니메이션은 동작하지 않음 (현재 가이드의 파벙ㅂ이 animation을 활용한 방법)
  • 위 두 가지 방법 중 요건에 따라 선택하여 작업이 필요
